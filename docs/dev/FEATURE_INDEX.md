## Known Gaps (vs. Enterprise NGFW)

The following capabilities are **not currently implemented** but are on the long-term roadmap:

| Feature | Current State | Gap |
|---------|---------------|-----|
| Layer 7 DPI | JA3/JA4 fingerprinting | True application identification |
| SSL/TLS MITM | Passive observation | Active decryption |
| Identity-based Policies | IP/Zone-based | LDAP/AD/OIDC integration |
| SD-WAN | Static failover | Quality-based path selection |
| Threat Intel Feeds | Static blocklists | STIX/TAXII automation |
| ZTNA | WireGuard VPN | Identity-aware proxy |
